# Weather-PowerBI-Dashboard
# 🌦 Real-Time Weather Dashboard - Power BI

## 📌 Project Overview
This project is a real-time weather analytics dashboard built using Power BI.  
It uses weather API data to visualize temperature, humidity, wind speed, and weather conditions.

## 🎯 Objective
The objective of this project is to analyze live weather data and present it in an interactive dashboard.

## 🛠 Tools Used
- Power BI
- Weather API
- Power Query
- Data Visualization

## 📊 Dashboard Features
- Real-time weather data
- Temperature analysis
- Humidity analysis
- Wind speed tracking
- City-wise weather comparison
- Interactive visuals
